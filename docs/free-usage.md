## Tips for Minimizing Costs

- Use **Sonnet** instead of Opus — significantly cheaper per token and handles most coding tasks well
- Add a `.claudeignore` file to exclude large files, build directories, and `node_modules` from context
- Use concise, specific prompts to reduce input token consumption
- Consider **Google Cloud Vertex AI** ($300 free credits for new accounts) or **AWS Bedrock** free tier as Claude Code backends
